NIIT OJ 导航助手 - 安装说明

1. 打开 Microsoft Edge 浏览器。
2. 在地址栏输入 `edge://extensions/` 并回车。
3. 在左侧菜单底部，开启 "开发人员模式" (Developer mode) 开关。
4. 点击顶部的 "加载解压缩的扩展" (Load unpacked) 按钮。
5. 选择本文件夹 (`f:\Desktop\niitOJutils\niit-oj-nav-extension`)。
6. 扩展加载成功后，刷新 NIIT OJ 的题目页面即可看到按钮。

注意：
- 如果您修改了代码，需要回到扩展管理页面点击 "刷新" (Reload) 按钮才能生效。
- 确保文件夹中包含 `manifest.json` 和 `content.js`。
